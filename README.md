# 豆格工坊

一个纯前端的图片转拼豆图纸工具。图片只在浏览器本地处理，不会上传。

## 本地运行

直接双击 `index.html` 即可使用。也可以在项目目录运行：

```powershell
python -m http.server 5173
```

然后访问 `http://127.0.0.1:5173`。

## 已实现

- 上传、拖拽 JPG / PNG / WEBP
- 自动推荐格数，手动选择 12–100 横向格
- 4–32 色近似拼豆色卡匹配
- 每格色号、横纵坐标、5 格加粗辅助线
- 浅色背景自动留空
- 色号 / 净图 / 原图三种预览
- 点击色卡后手动修正格子
- 每色数量、豆子总数、29×29 底板估算
- PNG 导出与打印图纸

> 内置色卡为通用近似色。不同品牌、批次和屏幕显示会有色差，正式备料前建议对照实体色卡。
